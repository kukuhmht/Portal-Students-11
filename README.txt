NOTE:
->Jangan merubah nama folder, karena harus merubah semua settingan yang ada di coding
->Database ini masih menggunakan phpmyadmin versi terdahulu yang belum support mysqli,
  Jika error db maka ubah settingan database nya di folder "application>config>database"
  dan cari "$db['default']['dbdriver'] = 'mysql';" jika tidak support 'mysql' maka ubah ke 'mysqli'
  *kalian bisa ubah sesuka hati kalian di settingan database di file tersebut
->Jika masih penuh error mungkin anda kurang tamvan :v
